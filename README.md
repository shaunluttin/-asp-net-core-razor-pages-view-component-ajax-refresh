
Start the app

    git clone https://github.com/shaunluttin/asp-net-core-razor-pages-view-component-ajax-refresh.git
    cd asp-net-core-razor-pages-view-component-ajax-refresh
    dotnet run

Then go to https://localhost:5001/index.html
